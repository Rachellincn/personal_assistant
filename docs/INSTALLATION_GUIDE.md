# 个人助手应用 APK 安装及使用教程

本项目已编译完成并生成了标准 APK 安装包，支持在小米 13 Ultra（Android 16）上运行。

## 1. 安装步骤

1.  **下载 APK**:
    *   将生成的 `app-debug.apk` 文件发送到手机上。
2.  **允许安装**:
    *   在手机上打开 APK 文件，系统会提示 "允许安装来自此来源的应用"。
    *   点击 "设置"，打开 "允许来自此来源" 开关，然后返回点击 "安装"。
3.  **运行应用**:
    *   在应用列表找到 "个人助手" 图标，点击运行。

## 2. 功能使用指南

### 记账模块
1.  **录入账单**:
    *   点击底部的 "记账" 按钮切换到记账模块。
    *   点击右下角的 "+" 按钮进入添加账单界面。
    *   输入金额（必填）、选择分类（默认餐饮）、选择日期并填写备注。
    *   点击 "保存" 即可。
2.  **编辑与删除**:
    *   点击列表项进行编辑。
    *   长按列表项进行删除。
3.  **统计分析**:
    *   点击左下角的 "查看统计" 按钮进入统计界面。
    *   可按日、月、季、年查看支出总额及分类占比图表。

### 日程模块
1.  **明日安排**:
    *   点击底部的 "日程" 按钮，选择 "明日安排" 标签。
    *   点击右下角的 "+" 按钮创建明日待办事项。
    *   输入描述、设置时间及优先级，点击 "保存"。
2.  **今日总结**:
    *   切换到 "今日总结" 标签查看当天的待办。
    *   勾选事项前的复选框，会弹出 "今日总结" 弹窗，输入简短总结后点击 "完成"。
    *   已勾选的事项会保留，作为当日完成的记录。
3.  **编辑与删除**:
    *   点击列表项进行编辑。
    *   长按列表项进行删除。

## 3. 注意事项

*   **权限需求**: 应用不需要任何敏感权限（如定位、相机、联系人等），仅使用本地存储。
*   **数据安全**: 所有数据保存在手机本地 SQLite 数据库中，卸载应用会导致数据清除。
*   **小米优化**: 针对骁龙 8 Gen 2 处理器进行了流畅度优化，且完美适配 Android 16 的 Material 3 设计语言。
